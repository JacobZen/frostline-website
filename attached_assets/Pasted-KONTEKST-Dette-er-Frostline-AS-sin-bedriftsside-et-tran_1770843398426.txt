KONTEKST: Dette er Frostline AS sin bedriftsside — et transportfirma i Nord-Norge.
Tech stack: React 18 + TypeScript + Vite + Express + Tailwind CSS v3 + Drizzle ORM + Neon DB.
Prosjektet er laget i Replit og deployes derfra.

OPPGAVE: Utfør følgende opprydding uten å endre funksjonalitet:

1. PACKAGE.JSON:
   - Endre "name" fra "rest-express" til "frostline-website"
   - Fjern disse UBRUKTE dependencies (verifiser med grep først at de ikke importeres noe sted):
     * @octokit/rest (GitHub API — ikke relevant for en transportside)
     * passport, passport-local (auth aldri implementert)
     * connect-pg-simple (session store — ikke brukt)
     * express-session (sessions — ikke brukt)
     * memorystore (session store — ikke brukt)
     * react-icons (bruk kun lucide-react som allerede brukes via shadcn)
     * tw-animate-css (duplikat av tailwindcss-animate)
     * input-otp (OTP-input — ikke relevant)
     * cmdk (command palette — ikke relevant for bedriftsside)
     * react-resizable-panels (ikke brukt)
     * vaul (drawer — sjekk om brukt)
     * embla-carousel-react (carousel — sjekk om brukt)
     * recharts (grafer — sjekk om brukt)
     * react-day-picker (datovelger — sjekk om brukt i booking)
   - Flytt @jridgewell/trace-mapping fra dependencies til devDependencies
   - Fjern @tailwindcss/vite fra devDependencies (v4-plugin som ikke brukes i vite.config.ts)
   - Fjern @types/connect-pg-simple, @types/passport, @types/passport-local (følger med pakkene over)

   VIKTIG: Bruk grep/søk for å sjekke at hver pakke FAKTISK ikke importeres før du fjerner den.

     plugins: [
       react(),
